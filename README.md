# parserx
lexer &amp; parser generator and toolkit for java

## Features
- accepts regex like grammar(EBNF)
- epsilon removal
- left recursion removal
- ebnf to bnf and vice versa
- LR(0),LR(1),LALR(1) dfa and table output as dot graph
- dot graph of NFA,DFA
- DFA minimization
- lexer generator
